# TaskFlow
## Summary
Aplicação de To-Do List desenvolvida como parte do Desafio IA no Desenvolvimento de Software,
com foco em Engenharia de Prompt aplicada ao ciclo de vida do software.
---
## Features
- Adição de tarefas com descrição personalizada
- Seletor de prioridade: Alta, Média e Baixa
- Estilização condicional por prioridade (cores e badges)
- Marcação de tarefas como concluídas
- Remoção de tarefas com animação de saída
- Contadores automáticos por nível de prioridade
- Animações fluidas com GSAP (entrada, conclusão, remoção)
- Interface dark mode, moderna e responsiva
